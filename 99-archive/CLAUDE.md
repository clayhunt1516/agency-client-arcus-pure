# 99 — Archive

> **For Claude:** Read this file before working in this folder.

## What this folder is

This folder holds superseded content kept for reference: old versions of SOPs, outdated reports, deprecated structure from earlier in the engagement. Content here is read-only.

## When to read content here

Only when explicitly asked, or when a specific archived item is needed for historical context. Do not routinely include archive content in your context — it is outdated by definition.

## Conventions

- Content is never deleted from this folder; it is added when something is deprecated elsewhere.
- Files here are named with the date they were archived (e.g., `sop-ap-processing-archived-2026-05.md`).
- Never edit archived files. If a deprecated item needs correction, note the issue in the current version of the relevant SOP or report, not here.
